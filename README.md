# GAN

https://zhuanlan.zhihu.com/p/41993080

使用方法：
1. 下载MNIST数据集：http://yann.lecun.com/exdb/mnist/
2. 创建MNIST_data文件夹，并放入解压后的数据
3. 运行脚本
